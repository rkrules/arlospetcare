# Icons Directory

Place custom SVG icons here if needed.

## Default Icons

The site uses Font Awesome 6 for icons, which provides thousands of free icons via CDN.

## Custom Icons

If you need custom icons:

1. Export as SVG from your design tool
2. Place in this directory
3. Reference in HTML: `<img src="assets/icons/your-icon.svg" alt="Description">`

## SVG Optimization

Use [SVGOMG](https://jakearchibald.github.io/svgomg/) to optimize SVG files before adding them.
